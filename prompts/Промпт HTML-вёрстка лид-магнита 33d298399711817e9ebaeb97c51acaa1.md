## Промпт

```jsx
Создай HTML-страницу лид-магнита для сервиса GoBanana (gobanana.ru).

Входные данные:
- Слаг страницы: [ВСТАВЬ, например: white-bg-wb]
- Заголовок: [ВСТАВЬ ЗАГОЛОВОК]
- Метаописание (до 160 символов): [ВСТАВЬ]
- Готовый текстовый контент: [ВСТАВЬ ПОЛНОСТЬЮ]
- Фотографии-примеры генерации: [ВСТАВЬ ССЫЛКИ или напиши «заглушки»]
- Ссылки на другие лид-магниты для перелинковки: [ВСТАВЬ]

---

ДИЗАЙН И СТИЛЬ:

Цвета (строго соблюдать):
- Фон страницы: #181613
- Фон карточек/блоков: #232018
- Фон карточек вторичных: #2A2620
- Акцентный (кнопки, хайлайты): #E8846A
- Акцентный при hover: #D4755C
- Текст основной: #FFFFFF
- Текст вторичный: #9A9490
- Текст третичный: #5C5854
- Граница блоков: #302C28
- Жёлтая точка логотипа: #F5C842

Логотип: слово «gobanana» строчными буквами, после него точка цвета #F5C842. Шрифт логотипа — такой же как весь текст страницы.

Шрифт: Inter через Google Fonts. Фолбек: -apple-system, BlinkMacSystemFont, sans-serif.
- Заголовок H1: 32-40px, font-weight 800, line-height 1.15
- Заголовок H2: 22-26px, font-weight 700
- Заголовок H3: 18px, font-weight 600
- Тело текста: 16px, line-height 1.65, font-weight 400
- Мелкий текст: 14px, цвет #9A9490

Адаптивность:
- Mobile first
- Брейкпоинты: 375px (мобильный), 768px (планшет), 1200px (десктоп)
- Максимальная ширина контента: 720px, центрирование
- Padding контейнера: 20px мобайл / 40px десктоп

СТРУКТУРА СТРАНИЦЫ (строго по порядку):

**1. ШАПКА**
Липкий header (position: sticky, top: 0, z-index: 100, backdrop-filter: blur(12px), фон с прозрачностью #181613CC).
Внутри: логотип слева, справа — три элемента в ряд:
- кнопка «Поделиться» (класс .btn-ghost .btn-icon) — onclick sharePage(), скрыта на мобайле (display:none), показывается от 640px
- кнопка «PDF» (класс .btn-ghost .btn-icon) — onclick window.print(), скрыта на мобайле, показывается от 640px
- кнопка «Попробовать бесплатно» — акцентная, со ссылкой https://gobanana.ru на новую вкладку, видна всегда

Класс .btn-ghost — общий для обеих серых кнопок (transparent background, border 1px solid var(--border), border-radius 50px, цвет var(--text2)). Класс .btn-icon добавляет display:none на мобайле и display:inline-flex от 640px.

**2. ГАЛЕРЕЯ-БЕГУЩАЯ СТРОКА**
Горизонтальная прокрутка фотографий-примеров генерации. Автоматически едет слева направо (CSS animation, infinite), пауза при наведении. Фотографии квадратные или 3:4, высота 200px, border-radius 12px, небольшой gap между ними. При клике на фото — лайтбокс (модальное окно с затемнением фона, закрытие по клику или Escape). Фотографии брать из входных данных. Если ссылок нет — создай заглушки div с градиентом и текстом «Пример генерации».

**3. ЗАГОЛОВОК И ПОДЗАГОЛОВОК**
H1 с заголовком. Под ним — вводный абзац (hero-lead, цвет var(--text2), font-size 17px). Под абзацем — hero-meta: строка с 3 бейджами (пилюли: фон var(--card), border, border-radius 50px) — например «5 промптов», «15 сек», «от 9 ₽». Под бейджами — hero-actions: блок с двумя кнопками «Поделиться» и «Сохранить PDF» (класс .btn-ghost), display:flex на мобайле, display:none от 640px. Это мобильные аналоги кнопок из шапки. Отступ hero сверху 40px.

**4. ОГЛАВЛЕНИЕ (Table of Contents)**

Логика: автоматически собирается из всех H2 и H3 заголовков страницы через JS при загрузке (querySelectorAll h2, h3). Каждому заголовку генерируется id из его текста (кириллица допустима через encodeURIComponent). Оглавление — прокручиваемые ссылки с плавным переходом (scroll-behavior: smooth).

Поведение на десктопе (>= 1200px):
- Боковая панель position: sticky, top: 80px (под шапкой)
- Ширина: 220px, отступ от неё до основного контента: 40px
- Фон: #1E1B17, border-radius 12px, padding 20px, border: 1px solid #302C28
- Заголовок «Содержание» — 13px, font-weight 600, цвет #9A9490, margin-bottom 12px
- Ссылки: ul без маркеров, 14px, цвет #9A9490, padding 6px 8px, border-radius 6px
- H2 — обычный отступ. H3 — padding-left: 16px (вложенность)
- Активный раздел — цвет #FFFFFF, фон #2A2620, border-left: 2px solid #E8846A
- Активный раздел обновляется через IntersectionObserver при прокрутке
- Максимальная высота: calc(100vh - 120px), overflow-y: auto, тонкий кастомный scrollbar

Layout на десктопе (CSS Grid):
```

.page-layout {

display: grid;

grid-template-columns: 220px 1fr;

gap: 40px;

max-width: 1000px;

margin: 0 auto;

}

.toc-sidebar { position: sticky; top: 80px; align-self: start; }

.main-content { max-width: 720px; }

```

Поведение на мобайле / планшете (< 1200px):
- Боковая панель прячется: display: none
- Вместо неё — аккордеон ПЕРЕД основным контентом (после заголовка и подзаголовка)
- Аккордеон sticky на мобайле: position: sticky, top: 60px (под шапкой), z-index: 50, background: var(--bg), padding: 8px 0 0 — прилипает при скролле
- Кнопка «Содержание ▾» — фон #232018, border-radius 12px, padding 14px 20px, font-weight 600, стрелочка справа, поворачивается при открытии (▴)
- При клике — список разделов разворачивается с анимацией (max-height transition 0.3s ease)
- Внутри те же ссылки, H3 с отступом. При клике — аккордеон закрывается, скролл к разделу
- Активный раздел — такой же стиль как на десктопе

Layout на мобайле:
```

.page-layout { display: block; }

.toc-sidebar { display: none; }

.toc-accordion { display: block; }

```

JS для оглавления (DOMContentLoaded):
1. querySelectorAll('.main-content h2, .main-content h3') — собирает заголовки
2. Каждому присваивает id из textContent (пробелы → дефисы, строчные)
3. Генерирует li > a[href="#id"] в оба контейнера — sidebar и аккордеон
4. H3 получает класс toc-sub (padding-left в стилях)
5. IntersectionObserver — при входе заголовка во viewport ставит класс active на ссылку в обоих контейнерах
6. Клик по ссылке аккордеона — закрывает аккордеон (убирает класс open)

**5. БЛОКИ КОНТЕНТА**
Последовательно выводи все блоки из текстового контента:
- Обычный текст — в p тегах, отступы между абзацами 16px
- Нумерованные шаги — styled ol, счётчик в акцентном цвете, каждый пункт простой строкой (padding-left 28px, без карточек), цвет текста var(--text2)
- Маркированные списки — ul со styled li, маркер в акцентном цвете
- H2, H3 — как указано в шрифтах
- Разделитель между крупными блоками — тонкая линия #302C28

**БЛОК ЦА (кому это нужно):**
Список из 2-3 сегментов без карточек. Каждый сегмент:
- audience-who: font-weight 700, font-size 16px, цвет var(--text)
- audience-problem: font-size 15px, цвет var(--text2), margin-bottom 8px
- audience-solution: font-size 15px, цвет var(--accent), font-weight 600 (начинается с «→»)
- Между сегментами — тонкая линия var(--border)

**БЛОК ТРЕБОВАНИЙ МАРКЕТПЛЕЙСА (если есть в контенте):**
Сетка 2 колонки (мобайл — 1 колонка). Карточка: фон var(--card), border-radius 12px, padding 18px 20px. Метка (WB / Ozon) — font-size 13px, font-weight 700, цвет var(--accent), uppercase, letter-spacing 0.5px. Список пунктов: font-size 14px, цвет var(--text2), маркер «·» в акцентном цвете, ключевые значения bold белым.

**БЛОК СОВЕТОВ ПО ИСХОДНИКУ (если есть в контенте):**
Список из 4-5 советов. Каждый совет — flex-строка: иконка (emoji в блоке 36×36px, фон var(--card), border-radius 8px, flex-shrink 0) + текст справа (заголовок font-weight 700 + описание font-size 14px, цвет var(--text2)). Gap между советами 16px.

**5. БЛОКИ С ПРОМПТАМИ**
Каждый промпт — отдельный компонент:
```

<div class="prompt-block">

<div class="prompt-header">

Промпт #N — Название

<button class="copy-btn" onclick="copyPrompt(this)">Скопировать</button>

</div>

<div class="prompt-when">Когда использовать: ...</div>

<div class="prompt-text" data-prompt="...">текст промпта целиком</div>

<div class="prompt-result">[СКРИНШОТ РЕЗУЛЬТАТА — заглушка или img]</div>

</div>

```
Стиль блока: фон #232018, border-left 3px solid #E8846A, border-radius 12px, padding 20px. Кнопка «Скопировать» — маленькая, серая, при клике меняется на «Скопировано ✓» на 2 секунды. Текст промпта — моноширинный шрифт, размер 14px, цвет #D4D0CC, white-space: pre-wrap.

**6. БЛОК ЛАЙФХАКОВ И СОВЕТОВ**
Если есть нумерованные советы — каждый на отдельной карточке. Карточка: фон #2A2620, border-radius 12px, padding 18px 20px 18px 64px, position relative. Слева — абсолютно позиционированная цифра: font-size 28px, font-weight 800, цвет var(--accent), opacity 0.4. Заголовок совета: font-weight 700, font-size 15px. Текст: font-size 15px, цвет var(--text2).

**7. ЧЕК-ЛИСТ (если есть в контенте)**
Кастомные чекбоксы — квадрат с галочкой SVG в акцентном цвете, кликабельны (toggle класса checked), текст зачёркивается при отметке.

**8. БЛОК CTA (призыв к действию)**
Отдельный блок с фоном #232018, border-radius 16px, padding 32px, text-align center.
Внутри:
- Заголовок «Попробуй прямо сейчас» (H2)
- Подпись: «Первые генерации — бесплатно. Без подписки.»

- Под кнопкой мелко: «200к+ пользователей · 12M+ генераций»

**9. ПЕРЕЛИНКОВКА**
Заголовок «Ещё полезное по теме» (H3).
Сетка карточек 2 колонки (мобайл — 1 колонка). Каждая карточка: фон #232018, border-radius 12px, padding 16px 20px, заголовок лид-магнита, одна строчка описания, ссылка «Читать →» в акцентном цвете.

**10. ИСТОЧНИКИ (если есть)**
Список в виде нумерованного ol, мелкий шрифт 13px, цвет #5C5854, ссылки в акцентном цвете.

**11. ПОДВАЛ**
Логотип gobanana слева. По центру или справа: «© 2026 GoBanana» и ссылка на gobanana.ru. Фон #111008, padding 24px.

---

КНОПКИ ПОДЕЛИТЬСЯ И PDF:
Не fixed-панель. Размещаются в двух местах:
1. В шапке (десктоп ≥640px): два .btn-ghost.btn-icon перед кнопкой «Попробовать бесплатно»
2. Под hero (мобайл <640px): блок .hero-actions с двумя .btn-ghost — «↗ Поделиться» и «⬇ Сохранить PDF»

Поделиться: navigator.share() если поддерживается, иначе navigator.clipboard.writeText(window.location.href) + alert «Ссылка скопирована!».
PDF: window.print().

---

LIGHTBOX (для галереи):
JS функция openLightbox(src). Модалка с фоном rgba(0,0,0,0.92), изображение по центру, max-width 90vw, max-height 90vh, object-fit contain, border-radius 12px. Крестик закрытия сверху-справа. Закрытие по клику вне фото или Escape.

COPY ПРОМПТА:
JS функция copyPrompt(btn). navigator.clipboard.writeText(текст из data-prompt). Меняет текст кнопки на «Скопировано ✓» с зелёным цветом на 2 секунды.

---

OPTIMIZATION:
- Только один внешний запрос: Google Fonts Inter
- Никаких фреймворков, никаких CDN библиотек
- Весь CSS inline в <style>
- Весь JS inline в <script> в конце body
- Изображения: loading="lazy", width и height прописаны
- Meta теги: charset, viewport, title, description, og:title, og:description, og:image (заглушка), og:url, canonical
- Print styles: убрать шапку, галерею, fixed панель; промпты печатать на белом фоне

---

ВЫВОД:
Один полный HTML-файл. Без пояснений, без markdown-обёртки — только чистый HTML от <!DOCTYPE html> до </html>.
```
